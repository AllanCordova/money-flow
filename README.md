# 💰 Money Flow

**Money Flow** é um sistema simples para **registro de transações financeiras**, contando com funcionalidades de **CRUD, filtragem avançada** e **validação de login**. O projeto foi desenvolvido com foco na **modularidade e escalabilidade**, utilizando boas práticas e recursos avançados do React.

## 🌐 Acesse o Sistema

[Acesse o sistema aqui](https://allancordova.github.io/money-flow/)

## 🚀 Funcionalidades

✅ **CRUD Completo** → Criação, leitura, atualização e remoção de transações.  
✅ **Filtragem Inteligente** → Permite buscar transações de forma eficiente.  
✅ **Validação de Login** → Autenticação simples para gerenciar usuários.  
✅ **Arquitetura Modular e Escalável** → Código organizado para fácil manutenção e expansão.  

## 🛠️ Tecnologias Utilizadas

- **React com TypeScript** → Componentização e tipagem segura.  
- **React Context API + Reducer** → Gerenciamento de estado global eficiente.  
- **React Hook Form** → Melhor desempenho e validação simplificada nos formulários.  
- **Bootstrap** → Estilização responsiva e moderna.  
- **React Router** → Navegação entre páginas sem recarregar.  

## 📦 Instalação e Execução

1️⃣ **Clone o repositório:**  
```sh
git clone https://github.com/AllanCordova/money-flow.git
```
2️⃣ **Acesse a pasta do projeto:**  
```sh
cd money-flow
```
3️⃣ **Instale as dependências:**  
```sh
npm install
```
4️⃣ **Inicie o projeto:**  
```sh
npm run dev
```
5️⃣ **Acesse no navegador:**  
```
http://localhost:5173
```


## 🎯 Sobre o Projeto

O **Money Flow** foi desenvolvido pensando em **boa organização do código**, utilizando conceitos modernos como **React Context API e Reducer** para o gerenciamento de estado, garantindo escalabilidade. Além disso, a utilização do **React Hook Form** melhora a performance dos formulários.
